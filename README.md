# 🤖 OTMAN BOT MANAGER

نظام استضافة متخصص لبوتات التليجرام مع لوحة تحكم احترافية لإدارة البوتات وإعداداتها.

## ✨ المميزات الرئيسية

- **لوحة تحكم الأدمن**: إنشاء حسابات مستخدمين وتحديد عدد البوتات المسموح لكل مستخدم
- **لوحة تحكم المستخدم**: إنشاء بوتات مستنسخة وتعديل إعداداتها بسهولة
- **إدارة ديناميكية**: تعديل الرسائل والأوامر والإعدادات دون الحاجة لتعديل الكود
- **تشغيل/إيقاف فوري**: تشغيل وإيقاف البوتات مباشرة من لوحة التحكم
- **مراقبة السجلات**: عرض سجلات البوت في الوقت الفعلي
- **واجهة ملء الشاشة**: تصميم responsive يملأ كامل عرض وارتفاع الشاشة

## 📋 متطلبات التشغيل

- Python 3.11+
- Flask
- psutil
- aiohttp
- وغيرها (انظر requirements.txt)

## 🚀 التثبيت والتشغيل

### 1. تثبيت المكتبات المطلوبة

```bash
sudo pip3 install psutil aiohttp requests httpx google protobuf pycryptodome PyJWT urllib3 protobuf-decoder pytz cfonts flask
```

### 2. تشغيل التطبيق

```bash
cd /home/ubuntu/otman_manager/host
python3.11 bot.py
```

سيعمل التطبيق على المنفذ `21047`

### 3. الوصول إلى لوحة التحكم

افتح المتصفح وادخل إلى:
```
http://localhost:21047
```

## ⚙️ الإعدادات الأساسية للبوت

عند إنشاء بوت جديد، يمكنك تعديل الإعدادات التالية:

### بيانات الحساب (مطلوبة)
- **معرف الحساب (UID)**: معرف حسابك في اللعبة
- **كلمة مرور الحساب**: كلمة مرورك المشفرة

### معلومات البوت
- **اسم البوت**: الاسم الذي سيظهر للمستخدمين
- **اسم المالك**: اسم صاحب البوت
- **حساب التيليجرام**: حسابك على التيليجرام
- **معرف الأدمن**: معرف الشخص الذي يمكنه إدارة البوت

### الرسائل
- **رسالة المساعدة**: الرسائل التي تظهر عند استخدام أمر /help
- **رسالة الأدمن**: الرسالة التي تظهر عند استخدام أمر /admin

## 🔑 API Endpoints

### للمستخدمين
- `GET /api/user/info` - معلومات المستخدم الحالي
- `GET /api/bots/list` - قائمة البوتات
- `POST /api/bots/create` - إنشاء بوت جديد
- `POST /api/bots/start` - تشغيل بوت
- `POST /api/bots/stop` - إيقاف بوت
- `POST /api/bots/delete` - حذف بوت
- `GET /api/bots/config/get/<folder>` - الحصول على إعدادات البوت
- `POST /api/bots/config/save/<folder>` - حفظ إعدادات البوت
- `GET /api/bots/logs/<folder>` - الحصول على سجلات البوت

### للأدمن
- `GET /api/admin/users` - قائمة المستخدمين
- `POST /api/admin/users/create` - إنشاء مستخدم جديد
- `POST /api/admin/users/delete` - حذف مستخدم
- `GET /api/admin/stats` - إحصائيات النظام

## 📝 ملاحظات مهمة

1. **معرف الحساب وكلمة المرور**: مطلوبة لتشغيل البوت، لا يمكن تشغيل البوت بدونهما
2. **الحد الأقصى للبوتات**: يتم تحديده من قبل الأدمن عند إنشاء الحساب
3. **صلاحية الحساب**: تنتهي صلاحية الحسابات العادية بعد المدة المحددة
4. **السجلات**: يتم حفظ سجلات البوت تلقائياً في ملف `server.log`

## 🛠️ استكشاف الأخطاء

### البوت لا يعمل
- تحقق من معرف الحساب وكلمة المرور
- تأكد من أن الحساب نشط وليس محظوراً
- اطلع على السجلات للمزيد من المعلومات

### لا يمكن الوصول إلى لوحة التحكم
- تأكد من تشغيل التطبيق
- تحقق من رقم المنفذ (21047)
- تأكد من عدم وجود جدار ناري يحجب المنفذ
